# Files And Usage Limits

## Supported Upload Types
Keywords: upload, file, pdf, pptx, docx, document
Users can upload common study document formats for extraction and AI-assisted analysis.

Typical supported types include:
- PDF
- PPTX
- DOCX
- plain text documents

## File Processing
Keywords: extraction, parse document, analyze file, context file
Uploaded files are processed so the AI Tutor can answer questions using user-provided learning material.

## Practical Limits
Keywords: limits, size limit, payload, timeout, long file
Some requests may be constrained by:
- file size limits
- request timeout limits
- model token/context limits

If users hit limits, recommended actions are:
- split very large files into smaller parts
- ask narrower questions
- retry when network/provider load is high
